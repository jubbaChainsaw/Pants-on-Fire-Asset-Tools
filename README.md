# Pants on Fire! Asset Tools

Standalone admin tooling for the Pants on Fire project.

## Repository structure

- `Admin Tools/`
  - `Pants-on-Fire-Card-Generator/` - local-first admin app for:
    - prompt matrix generation
    - strict one-card artwork rules
    - round-structured card text generation
    - deck prep exports (PNG, PDF, ZIP)
    - theme and round-type editing
    - import/export and localStorage persistence

## Notes

- This repository is tooling-only and should remain separate from runtime game code.
